# ✈️ India Flight Delay Intelligence Dashboard

🚧 **Project Under Development**

I am currently working on an end-to-end Data Analytics project focused on analyzing flight delays, airline performance, passenger traffic, and operational efficiency in India's aviation sector.

## 🔍 Project Goals

* Analyze airline On-Time Performance (OTP)
* Identify delay patterns and trends
* Evaluate airline reliability and efficiency
* Explore passenger traffic and route performance
* Build an interactive Power BI dashboard
* Generate actionable business insights from aviation data
